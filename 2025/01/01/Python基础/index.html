常用函数

abs（）：取绝对值

round（）：四舍五入

max（）：取最大值 max(2,3,4,5)

2. 字符串操作

  • print()：打印字符串
  • +：拼接字符串
  • *：重复几次字符串
  • len()：字符串长度
  • split()：切分字符串，参数为分隔符
  • join() ：将list合并为str
  • .replace()：替换字符串（需要重命名）
  • .upper()： 转换为大写
  • .lower()：转换为小写
  • .strip()：去掉空格
    • .lstrip()：去掉左边空格
    • .rstrip()：去掉右边空格
  • .format：格式化字符串

3. 索引

[]：可以用来对数组和字符串进行操作，可以进行切片操作。如果进行切片操作，是前闭后开。

4. 列表 list

列表是最常用的Python数据类型，它可以作为一个方括号内的逗号分隔值出现。列表的数据项不需要具有相同的类型。不限制长度。

创建

操作

.len()：求list长度

+：拼接list

*：将原来的list复制多少倍

[]：索引操作（切片）。可以叠加进行二位操作。

=：在某个位置配合[]使用。就是替换。

delete[n]：删除索引为n的元素。

[]：中括号中也可以进行切片操作。

in：判断元素是否是在list中。not in：是不在。也可以判断子串。

.count()：在list中参数出现了几次。

.index()：找到相应元素的位置索引。

.append()：在尾部添加元素。每次只能添加一个元素，元素可以是list。

.insert(index, element)：在index出插入element。

.remove(element)：删除元素，参数是元素。重复元素删除第一个。

.pop(index)：弹出元素。返回弹出元素，并且在list中删除元素。

.sort()：排序操作。

sorted()：可以进行赋值，并且不改变原始数据。

.reverse()：倒置列表。

5. 字典

one = {}，是一种键值对。

创建

操作

key - value

取值：

[key]：直接获取某个key对应的value。

赋值：

[key] = newvalue ：直接将某个key对应的value修改为newvalue。

字典和列表一样可以套娃。

.get(key)：类似于[]的操作，获取某个key对应的value。第二个参数可以自己设定如果没有key则返回的元素。

.pop()：弹出，类似list中的 pop（）。

.update()：更新dict()。

in：判断key是否在字典中。返回 bool值

.keys()：打印字典key

.values()：打印value

.items()：打印字典的元素

6. 集合 set

set：会保留唯一元素，自动去重。

one = {1,1,2,3,4} # one = {1,2,3,4}

.union()：求两个set的并集。也可以用管道符|来做相同操作。

.intersection()：求交集，&。

.difference()：找不同，返回set。-也可以。

.issubset()：判断是否是子集。<=,<,>

.add()：添加一个值

.update()：用一个list来修改set

.remove(index)：删除index处的元素

.pop()：弹出元素
