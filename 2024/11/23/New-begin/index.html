New begin

Created2024-11-23|Updated2024-11-23|随笔
|Post Views:

时光匆匆，大学四年转眼即逝，如今的我也面临着上研究生和工作的抉择。虽然一直想要上一个平台学习，但是现实往往不如意，考研一战的失利让我在这个选择上徘徊很久，我到底是否真的适合继续学习？还记得当初选择计算机这个专业是因为它是个不错的专业再者我也比较感兴趣，但是现在回看，经历过考研的折磨，我开始思考我是否真的热爱它？也许这种迷茫只是在长时间脱离社会脱离技术后一种趋利避害的反应，也许是我真的并不是完全热爱这个专业吧，在这里我并不想过多思考。思考过去的路就像是咀嚼吃过的馒头，苦的还是苦的。

当下，我继续我的考研之路，没错，这是二战了。

如今已经是11月份，离今年的考试还剩不到一个月的时间。我又开始迷茫，彷徨，甚至于害怕。这种长时间的付出并且可能颗粒无收的投资让我有点喘不过来气。

于是，我开始再次踏上网上技术冲浪。重新打开 GitHub 。我看了很多最新的技术，包括AIGC，我曾经喜欢的操作系统。甚至于我无意间发现了系统大佬的进阶之路。我又开始回想自己的大学时光。我曾经也搭过博客，但都是坚持不到一个月就弃之不理。以至于我现在什么技术的没学会，或者说入门。有点懊悔过去浪费的时间，但是又有什么用呢？我们只能把握当下和未来的事！

这次搭建这个博客我希望能重头再来。可以说是对过去的救赎，以及对未来的希望。

我才刚大学毕业，未来还有很大可能，加油！！！
