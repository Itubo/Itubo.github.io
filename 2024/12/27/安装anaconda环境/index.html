安装anaconda环境

Created2024-12-27|Updated2024-12-30|深度学习
|Post Views:

1. 背景

学习深度学习的相关知识需要用到Python以及一些相关的库，如果单纯一个个安装python不同的版本相对麻烦并且低效，由此Anaconda就产生了。

Anaconda中文大蟒蛇，是一个专为开发者和数据分析师设计的Python发行版本。作为一个功能强大且实用的数据科学平台，Anaconda集成了众多常用的数据科学工具和库，为数据分析、机器学习和科学计算提供了便捷的开发环境。

上面是Anaconda中文网的介绍，在这里我们可以简单把Anaconda理解成一个包管理工具。通过这个工具我们可以创建不同的环境，在不同的环境中分别下载不同的Python版本以及这个环境中需要的依赖，通过切换环境来快速使用不同的Python版本和在这个环境中的依赖。

2. 下载 Anaconda

首先，找到Anaconda的官网下载链接：https://www.anaconda.com/download，不需要创建账户，直接点击下面的***Skip registration***跳过注册环节。

进入后选择最新的Windows版本下载即可。

3. 验证及后续配置

3.1 Anaconda 的安装验证以及包的安装

打开开始菜单，找到Anaconda Prompt，点击后出现Anaconda的命令行工具。

这个base就是Anaconda默认的环境，我们可以通过输入conda list来查看默认的环境中安装的依赖包。同样的在这里可以通过Python的安装命令pip install xxx来安装依赖包。

如果遇上非官方工具包下载出错的问题，我们可以打开网站：https://www.anaconda.com/download

在这个非官方Python 工具包下载网站下载我们需要的工具包。这些工具包都是以whl结尾，也就是说我们可以直接通过pip命令安装。安装过程是切换到需要用工具包的环境中再进入whl文件所在目录，利用pip install xxx的形式安装。

3.2 Jupyter的使用

打开开始带单，找到刚才安装的Jupyter，点击Jupyter，会自动通过命令行显示创建一个本地的服务器。然后跳出来一个通过浏览器打开的网页，这个网页显示的就是本地的一些内容。

这个就是我们的工作目录。修改这个默认目录可以自己百度一下。
